# Create a cylinder

Create a sketch in the top plane			6\. Click Blind

Create a circle centered at the origin		7\. Select the contour \(the circle\)

Dimension the circle to be 1” in diameter 	8\. Extrude by 0\.35”

Click Features

Click Extrude Boss/Base
